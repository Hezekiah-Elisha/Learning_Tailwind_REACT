# Using TailwaindCSS

## Installation
### Create React App using Vite
```bash
npm create vite@latest client
npm install
```

### Install TailwindCSS
```bash
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
npx tailwindcss init -p
```

### creating router and pages
```bash
npm install react-router-dom
```
